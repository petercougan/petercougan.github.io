Choosing a first programming language

Introduction

When learning any new skill, one should base their approach around two things:

  • Fundamentals
  • Common Encounters

Learning to program is no different. For many people, programming seems very foreign, and deciding what to learn can be difficult enough.

Things to consider:

  • Language popularity
  • Generality
  • Level of abstraction
  • Familiar syntax
  • Typing system
  • Object-Oriented Capabilities
Language Popularity

There are far too many programming languages out there to even consider. Some are obviously much more used and known. This results in a larger community which leads to a more abundant supply of resources and documentation for learning. Thus, it is a no-brainer to only consider popular languages to start with.
